## Installation

### Requirements:

- Node JS: v20.0.0 to v21.0.0 (v21.0.0 not included)
- npm: v9.0.0 to v10.0.0 (v10.0.0 not included)
- Docker (Only for Docker Setup)

### Setup:

1. `npm install`
2. `npm run build`
3. `npm run start`

### Setup (Docker):

1. `docker build -t portfolio_builder .`
2. <code>docker run --rm -p 3000:3000 portfolio_builder</code> <br> OR <br> <code>docker run --rm -d -p 3000:3000 portfolio_builder</code> (to detach terminal)
